# Dice-Rolling-Simulator
Dice Rolling Simulator (Python)
# Overview
The Dice Rolling Simulator is a fun, interactive Python console application that simulates rolling a dice.
Each roll generates a random number between 1 and 6, displays a creative message using emojis, and automatically rolls again when a six is rolled.
This project is beginner-friendly and demonstrates core Python concepts in an engaging way.
